-----------------------------------------------
Instructions for installing the mxunit snippets
-----------------------------------------------


1) in Eclipse: go to Window -- Preferences -- CFEclipse
2) copy the path to the snippets directory
3) open that path on your file system
4) copy the snippets/mxunit directory into the snippets directory that you just opened
5) optionally, copy the contents of snippets/mxunit/keyCombos.properties into your existing keyCombos.properties file
