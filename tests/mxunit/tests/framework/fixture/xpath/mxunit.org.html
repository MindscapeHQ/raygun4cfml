    2. Get the MXUnit Eclipse Plugin
  • Download and install Eclipse and the MXUnit Framework
  • On the Eclipse menu, select Help > Software Updates > Find and Install. Select Search for New Features to Install. Click New Remote Site. Enter MXUnit Plugin for the Name and http://mxunit.org/update for the URL. Click Finish and follow the instructions.
  • Details: Open Eclipse, then read Help > Help Contents > MXUnit: ColdFusion Unit Testing Framework
  • System requirements: Eclipse 3.2 or later, running under JDK1.5 or later. YMMV with FlexBuilder standalone.
